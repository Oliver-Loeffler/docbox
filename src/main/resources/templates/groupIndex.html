Documentation artifacts are categorized similar to build artifacts or dependencies. An example would look like following:

Group:"com.amtc.metro.geva"
Artifact:"geva-core"
Version:"20210831.12"
Snapshot:"20210831173511655"

The "snapshot" value is derived from the upload timestamp including fractions of a second. This is how DocDrop handles multiple uploads with artifacts of the same version.
